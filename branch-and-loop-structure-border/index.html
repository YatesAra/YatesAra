1.1 条件结构🔀

条件结构一般会产生分支。

(可以按 含有Else 和 不含Else 分为两种形式。)对应的程序框图在于否后的流程线中有无另一步骤。

1.2 循环结构🔁

循环结构中反复执行的步骤称为循环体。

可以分为 当型循环结构 和 直到型循环结构。

循环结构中一定包含条件结构,这样才能判断改变的界限。
